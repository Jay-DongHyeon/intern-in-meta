# 상품 목록 페이지 요구사항 분석 및 기능 목록

## 📌 기본 정보
### 프로젝트명 : 슈킹 패션 쇼핑몰 상품 목록 페이지 (모바일)

### 사용하게 될 기술 : React

### 문서 작성자: 정동현

## 📝 고객 요구사항 정리
- 20~30대를 타겟으로 한 패션 브랜드 쇼핑몰 제작
- 브랜드의 주력 상품은 신발
- 개발 대상은 상품 목록 페이지

## 📋 기능 목록
- 모바일 환경에 최적화
- 상품 이미지 포함된 카드 형태 리스트
- 2개의 컬럼으로 상품 나열
- 장바구니 기능 : 
 - 상품 카드에 "담기" 버튼 제공
 - 버튼 클릭 시 장바구니에 항목 추가
 - 헤더(상단바)에 장바구니 개수 실시간 업데이트 표시
- 테스트 URL 클라이언트 제공
